# AGENTS.md

Repository workflow requirements for contributors and coding agents.

## Required maintenance

1. **Always update the app index**
   - Keep `INDEX.md` current whenever backend/frontend functions, routes, pages, or major behaviors change.

2. **Annotate functions and pages**
   - For each new/changed backend function or frontend page/module, document what it does in `INDEX.md`.

3. **Create and run tests after each change**
   - Add or update tests for behavior you changed.
   - Run relevant automated checks/tests before committing.
   - Include test commands and outcomes in your final summary/PR notes.

## Scope

These instructions apply to the entire repository rooted at this file.
