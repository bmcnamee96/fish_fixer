# Fish Fixer

React + Node migration workspace for the Fish Fixer application.

## Current structure

- `apps/api` - Node HTTP API scaffold for health, case analysis, chat, and photo endpoints.
- `apps/web` - React UI scaffold with Form, Chat, and Photo modes.
- `packages/shared` - Shared package placeholder for common schemas/types.
- `plan.md` - End-to-end migration and delivery roadmap.

## Run tests

```bash
pytest -q
```
